# README

## About

This is a [FileDB](https://github.com/ponmar/filedb2) demo configuration to play around with before creating one for your files collection.

The demo includes a collection of files (some example images), persons, locations and tags.

## Installation

1. Install [FileDB](https://github.com/ponmar/filedb2)
2. Clone this repository
3. Double-click on **demo\Demo.FileDB** to open the demo in FileDB
